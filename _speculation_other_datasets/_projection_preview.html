Projection preview

Same routes, two projections. Toggle to compare.

Each thumbnail plots only the GPS route, no photos, no playback. The dataset's bbox is fitted to the canvas so shape (not size) is what differs. With latfit off, every dataset is rendered in the Iceland-tuned 1:2 lon:lat ratio. With latfit on, longitude is rescaled by cos(centerLat)/cos(64.5°) so each dataset reads at its real geographic proportions. Iceland's cos(64.5°)/cos(64.5°) = 1, so the main piece is unchanged either way. Numbers in the meta line show the multiplier and the resulting width:height of the bbox.
